# 📇 Business Card Scanner App

A powerful Android app built with **Kotlin** and **Jetpack Compose** to scan business cards using AI, extract key contact details, and manage follow-ups with reminders and lead scoring.

---

## 🚀 Features

- 🔍 **OCR Business Card Scanner** using **Google ML Kit**
- 🧑‍💼 **Auto-Detect Name, Company, Role/Title, and Contact Details**
- ⏰ **Follow-Up Reminders & Notifications**
- 📊 **Lead Scoring System** for evaluating business contacts
- 📁 **Folder Management** for organizing scanned cards
- ❤️ **Favorites & Multi-Selection Actions**
- 📤 **Export Contacts to CSV / PDF**
- 🔒 **Secure Google & Email Authentication (Supabase/Firebase)**
- 📢 **AdMob Ads Integration** for monetization
- 🌙 Built with **Jetpack Compose**, **RoomDB**, and **MVVM Architecture**

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Room Database
- ML Kit Text Recognition
- Supabase Auth / Firebase Auth
- AdMob Ads SDK
- Notification Manager
- MVVM Architecture
